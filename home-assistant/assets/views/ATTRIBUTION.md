# Dashboard view photography

The processed dashboard banners are local derivatives of photographs published
under the Unsplash License:

- `services.jpg`: Tyler, “A rack of electronic equipment in a dark room”,
  https://unsplash.com/photos/OnI_TNcIv9U
- `casa.jpg`: Joao Vitor Torres, “Modern living room with dark furniture and
  planetary art”, https://unsplash.com/photos/7-9QPJAB6es
- `salud-v2.jpg`: image generated specifically for the Odín project using
  OpenAI image generation. It depicts a private home wellness scene with a
  smartwatch and pulse oximeter.
- `organizacion.jpg`: Jakub Zerdzicki, “Hand writing a checklist in a
  notebook”, https://unsplash.com/photos/TB7aNN4blTQ
- `monitor.jpg`: Neil Fernandez, “A modern laptop displaying a dark-themed
  dashboard”, https://unsplash.com/photos/6-0ajRI1cgs

The files were cropped, darkened where needed and compressed for use as Home
Assistant dashboard headers. Home Assistant stores its own uploaded copies, so
the live dashboard does not request external assets.
